🧰 Boîte à outils du Campaign Manager

Une sélection d'outils testés pour gagner du temps au quotidien ⏱️
Tous gratuits ou freemium.

💡 Cette liste évolue. Si vous connaissez un outil qui mérite d'y figurer, écrivez-moi.
MintyEmailTester Gratuit
mintyemailtester.com →
Envoie ton HTML sur une vraie boîte mail pour vérifier le rendu avant envoi. Simple, sans login, validation des alt tags incluse. Les 5 premiers envois sont gratuits, c'est peu mais ça peut dépanner.
Testi.at Freemium
testi.at →
Te génère une adresse de test, tu envoies depuis ton outil (Adobe Campaign, Brevo, Imagino…) vers cette adresse, et tu récupères des screenshots sur Gmail, Outlook, Apple Mail. Le rendu est donc celui de ton vrai routage. Plan gratuit limité à 3 screenshots/mois.
💡 Premier réflexe gratuit :
F12 dans Chrome → icône mobile → tu testes le responsive à toutes les largeurs, sans outil et sans limite.
Pour tester le dark mode : F12 → onglet Rendering → "Emulate CSS media feature prefers-color-scheme" → dark. Attention : ça simule le dark mode navigateur, pas le comportement des clients mail (Gmail, Outlook ont chacun leur propre logique).
⚠️ Pour aller plus loin (rendu réel dans Gmail, Outlook, Apple Mail), un outil dédié est nécessaire. Le marché du test gratuit s'est fortement réduit depuis l'arrêt de PutsMail. En pratique, passer par son ESP (Brevo, Adobe Campaign, Imagino) reste la solution la plus simple.
Pour une solution complète, je recommande⭐ Email on Acid → prévisualisation sur 100+ clients, test spam, vérification accessibilité et liens. À partir de 74$/mois (facturé annuellement). C'est cher, mais c'est vraiment la référence du marché.
iLoveIMG Gratuit
iloveimg.com →
Mon outil préféré pour tout ce qui touche aux images. Compression, rognage, redimensionnement, conversion, tout y est. Rapide et facile. La référence gratuite pour préparer ses visuels email.
Squoosh Gratuit
squoosh.app →
Idéal pour compresser une image en voyant la différence en temps réel avant de télécharger. Très utile pour affiner la qualité sur des visuels importants.
TinyPNG Gratuit
tinypng.com →
Efficace pour compresser en masse. Pratique quand on reçoit des lots de visuels trop lourds à préparer rapidement.
Mail Tester Gratuit
mail-tester.com →
Indispensable avant un routage important. Score de délivrabilité en 2 minutes, gratuit. Donne une vue rapide sur les problèmes SPF, DKIM et contenu.
MXToolbox Gratuit
mxtoolbox.com →
Vérifie les blacklists et la configuration DNS. Utile quand on reprend une infrastructure email existante pour s'assurer que le domaine n'est pas blacklisté.
GlockApps Freemium
glockapps.com →
Teste le placement réel de ton email boîte par boîte ( inbox, onglet Promotions ou spam ) chez Gmail, Outlook, Yahoo et d'autres. Utile quand tu as un problème de délivrabilité persistant et que tu veux comprendre pourquoi tu atterris en Promotions chez Gmail mais pas chez Outlook. 3 tests gratuits/mois.
Emaildojo Gratuit
emaildojo.io →
Prévisualise ton objet et ton préheader sur un large panel de clients ( iPhone, Gmail, Outlook desktop, Apple Mail, Samsung Mail, Thunderbird… Sans login ) sans limite.
TestSubject by ZURB Gratuit
zurb.com →
Ultra simple : colle ton objet, ton préheader et ton nom d'expéditeur, aperçu immédiat sur les principaux mobiles. Sans login, sans compte.
WAVE Gratuit
wave.webaim.org →
Extension Chrome gratuite pour détecter les problèmes d'accessibilité directement sur une page. Colle le HTML de ton email dans la version navigateur et WAVE t'indique les erreurs : alt text manquants, structure incorrecte, contrastes insuffisants.
Color Contrast Checker Gratuit
webaim.org →
Vérifie les ratios de contraste texte/fond. Indispensable pour valider la lisibilité en dark mode et pour les personnes malvoyantes. Ratio minimum : 4.5:1.
Can I Email Gratuit
caniemail.com →
Le "Can I Use" de l'email. Vérifie la compatibilité de chaque propriété CSS par client mail. Indispensable quand on intègre des éléments CSS avancés.
Stripo Freemium
stripo.email →
Builder drag & drop avec export HTML propre. Bien pour prototyper rapidement un template ou livrer un HTML à une équipe qui n'intègre pas en dur.
Beefree Freemium
beefree.io →
Interface intuitive, bon pour les équipes qui n'intègrent pas en dur. La version gratuite est suffisante pour un usage ponctuel.
Parcel Gratuit
customer.io/parcel →
Éditeur HTML email en ligne avec prévisualisation live, dark mode, vérification des liens et du spam score. Formatage automatique du code, détection d'erreurs, autocomplete email-specific. Gratuit, sans limite.
LanguageTool Freemium
languagetool.org →
Correcteur grammatical et orthographique disponible en extension Chrome et en ligne. Supporte le français, bien meilleur que le correcteur Word pour du contenu marketing. Indispensable avant de valider un BAT.
